# XY2APK - Konversi HTML ke APK Tanpa Login

Platform untuk mengkonversi file HTML/ZIP menjadi APK Android tanpa perlu login. Siap deploy ke Vercel.

## Fitur Utama
✅ Tanpa perlu login atau registrasi
✅ Upload icon aplikasi custom (PNG, JPG, GIF, WebP)
✅ Konfigurasi lengkap (nama, package, versi, izin)
✅ Proses cepat dengan teknologi serverless
✅ Gratis tanpa batasan
✅ Siap deploy ke Vercel

## Deploy ke Vercel

1. **Fork repository** atau buat project baru di Vercel

2. **Install Vercel CLI** (opsional):
```bash
npm i -g vercel
